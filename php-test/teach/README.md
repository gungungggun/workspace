## ステップ1
`jihan.php`をコマンドラインで
```
php jihan,php コーラ 1000
```
のように実行。  
そのとき、購入商品とお釣りを標準出力するプログラムにする。  
商品リストは`calc()`内に記述。  
関数の上にアノテーションコメントもつける。  
引数が逆だったり、買えない金額だったり、存在しない商品名だった場合の考慮はしない。
